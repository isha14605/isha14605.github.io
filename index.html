Vision Checker

At MedHacks 2020, I worked with a team to create a website that aims to ensure quality care during a global pandemic, by allowing patients to monitor their eye health through a virtual eye exam. I was responsible for the front-end development of our project, using HTML/CSS/JavaScript, Bootstrap, and Heroku, as well as integrating the recording functionality into the website. In addition, our team also used Google's Speech-to-Text API and Google Cloud in order to convert recordings of eye exams to text and store them, for processing through a scoring function that outputs a visual acuity score for the user. Our project was the winner of the Google COVID-19 Hackathon Fund and we aim to develop it further with improvements and new features.

Protein Inhibitor Pipeline

In the Dry Lab in iGEM Toronto this summer, I developed a machine learning model, built in Python, that uses a transformer architecture in order to generate inhbitor protein sequences for an input protein. Before curating a dataset and writing the model, I modeled coronavirus proteins using PyMol, a molecular visualization software. The dataset used consists of around 50,000 dimers from the Protein Data Bank (PDB). Numpy, Pandas, and TensorFlow were used to load the dataset from a .csv file and into our model, called GPTProtein, which is based off of GPT2, a transformer model, as mentioned earlier.

Euphoric Lyric

At Hack the 6ix 2020, my team and I worked on a machine learning model that takes in an audio sample as an input and outputs lyrics that corresponds to various timestamps. To do this, we used the DALI dataset, which contains hundreds of audio files, as our input. Before we could train the model, there were many steps required in pre-processing the audio files and JSON files that were provided in this dataset, primarily done through various Python functions and the audio2numpy library. To construct the model, we used Keras, Numpy, and TensorFlow.
